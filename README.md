# 📄 Resume Screening & Ranking System

## 📌 Overview
This project focuses on automating the resume screening process using Natural Language Processing (NLP). It ranks resumes based on their relevance to a given job description, helping recruiters quickly identify the best candidates.

---

## 🚀 Features
- Automated resume ranking
- NLP-based text analysis
- Semantic similarity matching
- Efficient candidate filtering
- User-friendly interface with Streamlit

---

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Transformers (SBERT)
- Streamlit

---

## 🤖 Methodology
- Used Sentence-BERT (SBERT) for generating embeddings
- Compared resumes with job descriptions using cosine similarity
- Ranked resumes based on similarity scores

---

## 📊 Workflow
1. Data Collection (Resumes & Job Description)
2. Text Preprocessing
3. Feature Extraction using Sentence Embeddings
4. Similarity Calculation (Cosine Similarity)
5. Ranking Output

---

## 📈 Results
- Successfully ranked resumes based on relevance
- Improved screening efficiency by automating manual filtering
- Demonstrated accurate semantic matching using SBERT
